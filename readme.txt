== Description ==

LC Testimonial Slider WordPress plugin is a great tool to show your clients testimonials. You can display testimonials in any post or page content simply by pasting shortcode in it in a clean, responsive and beautiful Slider format. The "LC Testimonials" are a Custom Post Type. You can enter the Name, Company, Website in the Custom Fields for the Testimonial and the actual Testimonial text in the "Description" and image as featured image.

= Features: =

* Testimonial jQuery Slider
* Autoplay option
* Simple Testimonial Management Interface
* Display Testimonials on any Page, Post
* Show "Category" specific Testimonials
* Shortcode Powered
* Custom Post Type
* Fully Responsive
* Major Browser Supported


== Installation ==

There are 3 ways to install this plugin:

= 1. The super easy way =
1. In your Admin, go to menu Plugins > Add
1. Search for LC Testimonial Slider
1. Click to install
1. Activate the plugin
1. A new sub-menu under Settings menu named `LC Testimonials` will appear in your Admin

= 2. The easy way =
1. Download the plugin (.zip file) on the right column of this page
1. In your Admin, go to menu Plugins > Add
1. Select the tab "Upload"
1. Upload the .zip file you just downloaded
1. Activate the plugin
1. A new sub-menu under Settings menu named `LC Testimonials` will appear in your Admin

= 3. The old way (FTP) =
1. Upload `lc-testimonials` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. A new sub-menu under Settings menu named `Testimonial Slider` will appear in your Admin

== Usage ==

* To insert the Testimonial Slider, paste the below shortcode in the content section of your WordPress post/page:

`[lc_testimonials]`

== Frequently Asked Questions ==

= How to auto slide Testimonial Slider =

On WordPress dashboard, Go to LC Testimonial Slider => Settings page. You will see option Auto Slide. You will find the checkbox next to it. Select the checkbox to auto slide the testimonial slider or uncheck for manual slide. Save Changes.

= How to show Pagination Links in  Testimonial Slider =

On WordPress dashboard, Go to LC Testimonial Slider => Settings page. You will see option Pager. You will find the checkbox next to it. Select the checkbox to add pager to the testimonial slider or uncheck to hide it. Save Changes.
